

# Rohati Ayurved Landing Page

A complete Ayurvedic clinic landing page built with React, Vite, and Tailwind CSS — converting all uploaded components from CSS to Tailwind styling.

## Sections (in page order)

### 1. Navbar
- Sticky header with logo, navigation links (About Us, Blogs, Our Centers, Testimonials, Contact)
- Scroll-aware background change (transparent → solid)
- Mobile hamburger menu

### 2. Hero Section
- Full-screen hero with decorative circles
- Headline: "Pure Ayurveda. Powered by Nature."
- Subtext about Ayurvedic manufacturing
- Feature cards: 100% Herbal, GMP Certified, Quality Tested, AYUSH Approved

### 3. Expertise Section
- Left side: list of expertise areas (Complex Infertility Cases, Ayurvedic Alternative to IVF, etc.)
- Right side: action cards (Book Appointment, Success Story, Locate Clinic, Contact Us)
- Bottom stats bar: 25+ Years, 25000+ Patients, 100% Natural, 2000+ Panchakarma, 93% Success Rate

### 4. Disease Carousel
- Horizontal scrollable carousel with cards for PCOD, Uterine Disorders, Male/Female Infertility
- Each card shows stats (15+ Specialists, 500+ Procedures)
- Left/right navigation arrows

### 5. Diseases Detail Section
- Two-column grid showing Female & Male Infertility conditions
- Separate section for Non-Infertility issues (Female & Male)

### 6. Ayurveda Scroll Section
- Sanskrit verses with translations about Ayurvedic approach to fertility
- Decorative scroll/parchment styling

### 7. How It Works
- 4-step process: Herb Selection → Traditional Processing → Quality Testing → Certified Delivery
- Icon cards with arrow connectors

### 8. Testimonials
- Carousel with patient success stories
- Each card: quote, title, description, author info, star rating, and image placeholder
- Navigation arrows

### 9. Footer
- 4-column layout: Quick Links, Treatments, Diseases, Contact
- Location tabs (Nashik, Thane, Pune)
- Social media icons
- Bottom bar with copyright & legal links

## Design Approach
- Earthy, warm Ayurvedic color palette (greens, oranges, warm neutrals)
- All styling via Tailwind CSS (no separate CSS files)
- `lucide-react` icons replacing `react-icons`
- Fully responsive (mobile-first)
- Placeholder images where originals aren't available
